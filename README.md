# Tamibara
게임형 타이머 어플

## 🖥️ 프로젝트 소개
목표를 달성하면 귀여운 캐릭터를 성장시킬 수 있는 게임형 타이머 어플입니다.
<br>
우리의 일상생활에서 빈번하게 사용되고 있는 타이머, 하지만 이런 타이머 앱들은 필요할 때만 사용되고 시간이 지나면 잊혀지는 경향이 있습니다.
<br>
게임 형식의 타이머 앱인 타미바라로 목표 달성에 대해 지속적으로 동기를 부여 받으며, 재미있게 목표를 달성해보세요!
